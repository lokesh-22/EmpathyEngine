# Empathy Frontend

This folder contains the React + TypeScript + Vite frontend for Empathy Engine.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## What it does

- collects input text from the user
- calls the backend `POST /speak` endpoint
- fetches the generated playback audio from `GET /get-audio`
- plays the result in the browser
